# Salman and Co. Static Website

This repository contains a premium static website for Salman and Co., a Chartered Accountant firm in Karnataka.

## Features
- Responsive multi-page static website
- SEO-ready structure with sitemap, robots.txt and schema markup
- Premium corporate design with gold, navy and white palette
- Contact, WhatsApp and call-to-action sections

## Deployment
Deploy directly to Netlify or GitHub Pages by uploading the repository contents.
